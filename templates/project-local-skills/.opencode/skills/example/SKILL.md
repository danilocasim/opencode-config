# Example Skill Router (Project-Local)

This is a template. Replace `example` with your real skill name and add leaf docs that encode your repo-specific conventions.

## When to load

- You want repo-specific rules that override global guidance.
- You need a consistent pattern for this codebase.

## When NOT to load

- The global skill already covers it and the repo has no special rules.
